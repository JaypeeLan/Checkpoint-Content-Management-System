# EduCMS Backend

## Features
- JWT auth + RBAC middleware
- Redis cache integration
- Elasticsearch full-text search integration
- Cloudinary signed upload URL flow
- Socket.IO for realtime dashboard events
- Rate limiting + Helmet + CORS + compression
- Swagger at `/api/v1/docs`
- Jest unit/integration tests

## Run
```bash
npm install
cp .env.example .env
npm run dev
```

## Test
```bash
npm test
```
